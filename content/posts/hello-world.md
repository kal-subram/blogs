This is the first post on this blog. It's written in Markdown, lives in a git
repo on my local machine, and gets published automatically by GitHub Actions
whenever I push to `main`.

## Why this setup

- Write posts locally in `content/posts/` as Markdown files.
- Commit and push to GitHub.
- A GitHub Actions workflow builds the site with Hugo and deploys it to
  GitHub Pages — free hosting, no server to manage.
